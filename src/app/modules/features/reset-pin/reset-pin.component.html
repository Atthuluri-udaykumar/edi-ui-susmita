Reset PIN

The system will issue a new PIN for the corresponding Account when the Reset action is selected. An email containing the new PIN will be sent to the Account Contact's email address of record.

Reset the PIN only if the following criteria are met:

  1. The caller has been validated as the Account's Contact following the BCRC Secure Web Site Authentication Guidelines for EDI Representatives and based on the User Information in the Web site.

    2. and

  2. The caller cannot state the correct PIN.

Warning: If the caller cannot be validated, request that the authorized Submitter/Contact for the account submit a written request addressed to the Benefits Coordination and Recovery Center on company letterhead. Notify the caller that the request may be faxed and supply the fax number.

If the caller is validated and states the correct PIN, use the Unlock PIN function.
