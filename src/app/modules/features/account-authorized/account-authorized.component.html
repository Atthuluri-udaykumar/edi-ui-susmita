Change Authorized Representative Query

From this screen, you can update personal information for an existing Authorized Representative or replace an existing Authorized Representative with a different person. You have the option to query by RRE ID or user E-mail Address for the Authorized Representative you wish to change or replace.

To change the Authorized Representative, enter either the RRE ID or the E-mail address of the Authorized Representative you wish to change and select the appropriate link.

IMPORTANT: Prior to processing a request, you must follow the COBSW Authentication Guidelines to verify the authority of the Requester for each EIN submitted or associated with this request.

OR

To update personal information for an existing Authorized Representative select the Update button.

To replace an existing Authorized Representative for one or more RRE's select the Replace button.

NOTE: If the value is entered in the RRE field AND the E-mail field, the query to process the RRE will be processed. To process the E-mail query only enter a value in the E-mail field.
